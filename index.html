Dropdown


Dropdown, ou menu suspenso, é um elemento de interface gráfica que exibe uma lista de opções após um clique em um botão ou elemento acionador, economizando espaço na tela ao ocultar a lista de itens até que sejam necessários. Esse recurso é amplamente utilizado em sites e aplicativos para selecionar valores de listas longas, como países em um formulário, ou para apresentar opções contextuais.
